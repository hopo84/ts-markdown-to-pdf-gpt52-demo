HTML 直接转 PDF 测试

简介

这是一个用于测试 HTML 直接转 PDF 功能的文档。程序将直接读取此 HTML 内容并转换为 PDF。

功能特性

  • 支持 Markdown：使用 markdown-it 渲染 .md 文件
  • 支持 HTML：直接读取 .html 文件内容
  • 代码高亮：基于 highlight.js
  • 中文支持：优秀的中文字体渲染

代码示例

function greet(name) {
    console.log(`Hello, ${name}!`);
    return true;
}

greet('World');

表格示例

功能 状态 说明
Markdown 转换 ✅ 已支持 使用 markdown-it 渲染
HTML 转换 ✅ 已支持 直接读取 HTML 内容
命令行参数 ✅ 已支持 支持 -f 和 --file 参数

引用块

提示：HTML 文件会保留原有的样式，但 PDF 生成时会应用项目的全局样式。

使用方法

# 处理 Markdown 文件
npm start -- -f 1.md

# 处理 HTML 文件
npm start -- -f test.html

总结

通过优化后的代码，现在可以同时支持 .md 和 .html 两种输入格式，提供更灵活的使用方式。
